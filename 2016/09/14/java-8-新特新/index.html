java 8 新特新

发表于 2016-09-14

前言

这些新特性在MyEclipse 14及之前的版本都不支持1.8的编译时环境(最高才1.7)。测试的时候可以用系统编译时环境（cmd），或者用idea，它完美支持1.8甚至1.9

接口的默认方法

java 8 允许我们在接口中添加默认实现了，只需要在方法签名中添加default关键字就可以了。

具体应用场景和动机博主尚不得知，还有待研究。

Lambda 表达式

“Lambda 表达式”(lambda expression)是一个匿名函数，Lambda表达式基于数学中的λ演算得名。

介绍JAVA中Lambda 表达式的应用之前，先介绍相关知识, 主要来自这位的博客，更详细的可以戳进去。
